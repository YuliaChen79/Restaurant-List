# 我的餐廳清單

## 功能描述
### 使用者需要先登入會員才可使用相關功能
* 自行註冊帳號密碼作為登入資訊
* 使用Facebook登入
### 餐廳清單相關功能
* 瀏覽所有餐廳
* 瀏覽餐廳詳細資訊
* 新增餐廳資訊
* 修改餐廳資訊
* 刪除餐廳資訊
* 輸入關鍵字可搜尋目標餐廳
* 透過Google map查看位置
* 為餐廳評分

## 安裝與執行步驟
1. 請先確認已安裝Node.js 與 npm
2. 開啟終端機(Terminal)，將專案clone至本地
```
git clone https://github.com/YuliaChen79/Restaurant-List
```
3. cd進入專案資料夾並輸入：
```
npm install
```
4. 執行nodemon app.js
```
nodemon app.js
```
5. 開啟瀏覽器進入下列網址
```
http://localhost:3000
```
6. 若要暫停運行請輸入
```
Ctrl + c
```
