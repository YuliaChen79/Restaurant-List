# 我的餐廳清單

## 開發工具
* Node.js v14.16.0
* Express v4.16.4
* Express-Handlebars v3.0.0
* Bootstrap v5.1.3
* Popper v2.9.1
* Font Awesome


## 安裝與執行步驟
1. 請先確認已安裝Node.js 與 npm
2. 開啟終端機(Terminal)，將專案clone至本地
```
git clone https://github.com/YuliaChen79/Restaurant-List
```
3. cd進入專案資料夾並輸入：
```
npm install
```
4. 執行nodemon app.js
```
nodemon app.js
```
5. 開啟瀏覽器進入下列網址
```
http://localhost:3000
```
6. 若要暫停運行請輸入
```
Ctrl + c
```


## 功能描述
* 查看所有餐廳
* 輸入關鍵字可搜尋目標餐廳
* 瀏覽餐廳的詳細資訊
* 透過Google map查看位置
